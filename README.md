# getversion


Add following code to end of configuration shell file (.bashrc or .zshrc)

```sh
source getversion.sh
```


Following code
```sh
$ getversion python node docker rust
```

then you can see

```sh
Package              | Version
---------------------|--------------------
python               | 3.8.7
docker               | 20.10.2
rust                 | 1.51.0-nightly
```
